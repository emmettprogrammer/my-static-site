Practice Your Smokes

The best way to practice throwing smokes is to hop into a casual match with no bots. First, enable console commands if you haven't done so already (Settings > Game > Enable developer console). Next, select "practice with bots" in the upper-left dropdown. Start a casual match with your map and select "no bots" in the upper-right hand corner.

Pull up your console in-game by pressing "~". You can enter the following commands in this terminal.

  • First, enable cheats.
    sv_cheats 1

  • Bind noclip to a key. The key 'p' is used in the first example, but I like to use the side mouse button, so I use example 2.
    bind p "noclip"
    bind "mouse4" "noclip"

  • Bind a key to give you smoke grenades. You can use any key but I use mouse5.
    bind "mouse_5" "give weapon_smokegrenade"

  • Alternatively you can allow yourself you buy anywhere and give yourself a bunch of money.
    mp_buy_anywhere 1; mp_buytime 60000; mp_startmoney 65535; mp_restartgame 1;

  • Finally, run this command to give yourself 60 minute rounds.
    mp_roundtime_defuse 60; mp_roundtime_hostage 60; mp_roundtime 60; mp_restartgame 1

All set! To find all CSGO commands, go here.

Some other useful commands:

  • Infinite ammo
    sv_infiniteammo 1
  • Change map to Nuke (for example)
    map de_nuke
